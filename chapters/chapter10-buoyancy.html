第 1 节

浮力：测力计示数为什么会减小？

浸在液体或气体中的物体受到竖直向上的浮力；称重法把“托力”转化成可测量的示数差。

称重法的受力链

  • 先在空气中测出物体重力 G。
  • 物体浸入液体后，受到向上的拉力 F示 和浮力 F浮。
  • 静止时 F浮 + F示 = G，所以 F浮 = G - F示。
  • 液体密度越大、V排越大，浮力越大。
  • 完全浸没后继续加深，V排不变，浮力通常不再变化。

“浸在液体中”既包括部分浸入，也包括完全浸没；正在下沉的物体同样受到浮力。

第 2 节

阿基米德原理：同时比较两组差值

F浮 = G排 = ρ液gV排。实验必须等溢流停止并稳定读数，才能正确比较。

验证步骤

  1. 测物体重力 G 和空桶重力 G桶。
  2. 物体浸入溢水杯，计算 F浮 = G - F示。
  3. 收集排开的液体，计算 G排 = G桶和液 - G桶。
  4. 比较 F浮 与 G排，多次改变 V排 或液体密度。

V排是物体排开液体的体积，部分浸入时小于物体体积，完全浸没时才等于物体体积。

第 3 节

浮沉条件：比较力，也可以比较平均密度

完全浸没后松手，ρ物 与 ρ液 的关系决定物体先上浮、悬浮还是下沉；达到最终平衡后还要重新分析受力。

四种状态

上浮
F浮 > G，物体向上运动
漂浮
F浮 = G，V排 < V物
悬浮
F浮 = G，V排 = V物
下沉
F浮 < G，沉底后还有支持力

状态判断不能停在“松手瞬间”

  • ρ物 < ρ液：物体先上浮，最终通常漂浮，且 F浮 = G。
  • ρ物 = ρ液：物体可悬浮，完全浸没且 F浮 = G。
  • ρ物 > ρ液：物体下沉；沉底后还受到支持力。
  • 空心物体应比较整体平均密度，不能只看材料密度。

改变密度，物体最终停在哪里？

完全浸没后松手
最终状态上浮后漂浮

ρ物 < ρ液，物体先上浮；漂浮时约有 80% 体积浸入液体。

轮船

空心结构增大排水体积

排水量是满载时排开水的质量；漂浮时，它等于船和货物的总质量。

潜水艇

改变自身重力

水舱进水使重力增大而下潜，排水使重力减小而上浮。

气球和飞艇

空气也会产生浮力

内部气体的平均密度小于空气，且空气浮力大于总重力时可以升空。

密度计

液体越密，露出越多

密度计始终漂浮，浮力不变；液体越密，所需 V排 越小。

第 4 节 · 跨学科实践

制作微型密度计：让小瓶酒精也能测

密度计漂浮时 F浮 = G，因此 V排 = m / ρ液。质量决定排开体积，细标度杆把很小的体积变化放大成容易观察的液面高度变化。

结构与设计约束

  • 标度杆：直径约 3 mm，越细越容易分辨密度变化。
  • 浮子：增大可排开液体的体积，使装置能够漂浮。
  • 配重：放在最下方，让密度计保持竖直稳定。
  • 总质量：约 1 g，避免在小瓶中排开过多酒精。
  • 消毒酒精密度在 0.86～0.88 g/cm³ 可作为合格范围。

密度越小，密度计需要排开更多液体，因此下沉更深；读数前要等待它停止晃动。
